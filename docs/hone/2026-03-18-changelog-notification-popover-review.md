#### 🟡 Non-goal blocks necessary library

**Location**: Non-goals section
**Question**: The spec excludes an external PT rendering library but requires rendering headings, bold, italic, lists, links, code. A custom renderer for this subset of Portable Text is non-trivial.

**Action**: Removed from non-goals. Library now permitted.

### Assumptions

#### 🟡 Version format mismatch risk

**Location**: How it works, Step 1-2; Sanity details, GROQ query
**Question**: The GROQ query does an exact string match on `version`. If the Sanity content editor types `v1.2.3` but the app reports `1.2.3`, the query returns nothing and the popover silently never shows.
**Finding**: Version format is not specified, creating a silent failure mode.
**Developer response**: Add format spec — semver without 'v' prefix.
**Action**: Added explicit assumption with Sanity field validation recommendation.
